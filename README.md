Super Tucano for MSX2+

## Development

* To build the game, run `make` from `src`. This will generate the stucano.rom
  file, which you can play using any MSX emulator or on real MSX hardware.

### Pasmo Assembler

Super Tucano was developed using Pasmo Z80 [cross-assembler](https://pasmo.speccy.org).

** Installing Pasmo on Debian/Ubuntu

```bash
sudo apt update
sudo apt install pasmo
```
